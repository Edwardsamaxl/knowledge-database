# 学习者画像

## 基础身份

- **身份标签**：大学生，技术实战派，AI 原生代创作者
- **核心定位**：用输出倒逼输入， All in AI 的长期主义者
- **个人 slogan**：真正的成长不是知道更多，而是能拆开了重组为自己的能力
- **学习场景**：
  - 碎片化时间刷 X / GitHub / 即刻 / 教程视频
  - 深度时间做项目、写长文、拍抖音教程
  - 所有输入最终必须流向 Obsidian 知识库或某个平台账号
- **时间特征**：课余时间碎片化，但进入心流后能连续几小时深度工作

## 学习模式

| 阶段 | 行为 | 痛点 |
|------|------|------|
| 输入 | 刷到好东西 → 收藏/截图/转发给自己 | 收藏夹吃灰，三个月后完全不记得 |
| 加工 | 手写原始思考，或让 AI 辅助整理 | 整理完不复习，和没学一样 |
| 输出 | X 短推试水温 → 反馈好的扩展成长文/脚本 | 经常开了坑不填，Day 系列断更 |
| 归档 | 丢进 Obsidian 某个文件夹 | 缺乏结构，检索困难，无法联动 |

**关键洞察**：他不是学得太少，是**学了之后没有闭环**。learner 的使命就是强制完成这个闭环。

## 当前在学的领域（动态更新）

- **AI 工具链**：Claude Code / Cursor / v0 / Replit Agent 等 vibe-coding 工具
- **内容创作**：X 运营、抖音口播脚本、个人 IP 冷启动
- **商业模式**：API 中转站、AI 套壳工具、信息差生意
- **编程实战**：Next.js / TypeScript / Python 自动化脚本
- **知识管理**：Obsidian 工作流、Zettelkasten 卡片盒笔记法

## 输出矩阵（学习成果的出口）

| 平台 | 形式 | 对学习的要求 |
|------|------|-------------|
| X | 短推/长文 | 能提炼出一个有观点的洞察 |
| 抖音 | Day 系列教程脚本 | 能把复杂概念降维到 60 秒口播 |
| Obsidian | 结构化知识笔记 | 建立可链接、可复用的知识节点 |
| GitHub | 开源项目/脚本 | 代码可运行，README 能说清楚原理 |

## 语言 DNA（学习场景中同样适用）

1. **拒绝学术腔，只要人话**
   - 正确："说白了就是给 prompt 加了个记忆层"
   - 错误："该技术通过引入上下文感知机制显著提升了对话连贯性"

2. **必须挂载具体项目/工具/场景**
   - 正确："用 Cursor 写这个脚本的时候，我卡在 agent 模式不会读取环境变量"
   - 错误："在使用 AI 辅助编程工具时可能会遇到一些配置问题"

3. **接受半成品和错误**
   - 不追求笔记完美，追求"明天还能看懂"
   - 允许"这个我目前理解可能是错的，先记下来"

## 学习红线（learner 不能做的事）

- ❌ 生成读书笔记式的摘要（"本章主要讲述了..."）
- ❌ 推荐泛泛的"10 个必学资源"列表
- ❌ 代替他思考（learner 只负责结构化和追问，不替他给结论）
- ❌ 制造学习焦虑（"你还有很多没学"）
- ❌ 生成无法链接到现有知识库的孤立笔记
