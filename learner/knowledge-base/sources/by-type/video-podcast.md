# 视频与播客源

> 视频和播客的链接容易失效或被删除，记录时必须保存：
> 1. 原始链接
> 2. UP主/频道名
> 3. 发布时间
> 4. 关键时间戳和摘要（防止视频下架后信息丢失）

## 格式模板

```markdown
### [标题]
- **平台**: YouTube / Bilibili / 小宇宙 / Spotify / 其他
- **URL**:
- **创作者**:
- **发布时间**:
- **获取时间**:
- **状态**: untriaged / processing / extracted / archived
- **时长**:
- **关键时间戳**:
  - 00:00 - 主题介绍
  - 05:30 - 核心观点
- **一句话摘要**:
- **是否值得二次输出**: 是 / 否
```
